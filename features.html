1 Critical — Daily-use gaps Reached for multiple times a day by every GitExtensions user
Feature GitExtensions Form Status Notes
File History
 FormFileHistory Done Right-click any file → full commit history for that path. Uses git log --follow -- <file>. Shows renames, per-file graph, diff per revision.
Blame / Annotate
 FormBlame Done Line-by-line git blame view with author, date, commit hash per line. Click a line → jump to that commit in the graph.
Partial Staging (hunks/lines)
 FormCommit Done Currently only whole-file stage/unstage. GitExtensions lets you stage individual diff hunks or cherry-pick specific lines. Requires git apply --cached with a patched hunk.
Conflict Resolution UI
 FormResolveConflicts Done After a failed merge/rebase, lists conflicted files with ours/theirs/base. Launch external 3-way merge tool or use built-in. Mark as resolved. Commit merge.
Rebase
 FormRebase Done git rebase <branch> with onto target, preserve-merges, and autosquash options. Separate from interactive rebase (Tier 2).
Remote Management (CRUD)
 FormRemotes Done Add, rename, remove remotes. Edit fetch/push URLs. Set default remote. Prune stale remote-tracking branches. Currently only fetch/pull/push exist — no way to manage remotes at all.
Reflog Viewer
 FormReflog Done git reflog shows every HEAD movement. Essential recovery tool. Checkout any reflog entry to restore lost work.
Ahead / Behind Counts in Sidebar
 FormBrowse Done Each branch in the sidebar shows ↑2 ↓3 relative to its upstream. Populated via %(upstream:track) in for-each-ref. Already partially available in ref data — just not displayed.
2 Important — Frequent power-user features Used weekly; significant workflow gaps without them
Feature GitExtensions Form Status Notes
Interactive Rebase
 FormRebase Done Reorder, squash, fixup, drop, edit, reword commits. Requires parsing git rebase -i todo list and rendering it as a draggable UI.
Stash Improvements
 FormStash Done Missing: pop stash, --include-untracked / --all flags, stash diff view (see what's in a stash before applying), stash branch.
Compare Branches / Commits
 FormCompareToBranch / FormDiff Done Diff any two refs (branches, tags, commits) against each other. Shows changed files + unified diff. Uses git diff <ref1>...<ref2>.
Format Patch / Apply Patch
 FormFormatPatch / FormApplyPatch Done Export commits as .patch files (git format-patch). Apply patch files (git am / git apply). Common in email-based workflows.
Checkout Remote Branch (with tracking)
 FormCheckoutBranch Done When checking out origin/feature from the remotes list, should offer to create a local tracking branch. Currently sidebar remote items have no action.
Set Upstream Tracking
 (branch management) Done git branch --set-upstream-to=origin/main. Needed to fix detached ahead/behind counts and make pull/push work on branches without tracking.
Prune Remote Branches
 FormDeleteRemoteBranch Done Delete remote-tracking refs that no longer exist on the server (git fetch --prune or git remote prune origin).
Submodule Management
 FormSubmodules / FormMergeSubmodule Done Add, init, update, sync, remove submodules. Show submodule status in graph. Repos with submodules are common in enterprise/monorepo setups.
Clean / Cleanup Repository
 FormCleanupRepository Done git clean -fd with a file preview before deletion. Remove untracked files/directories safely.
Add to .gitignore
 FormAddToGitIgnore Done Right-click untracked file in staging area → "Add to .gitignore". Appends pattern to .gitignore and refreshes status.
Edit .gitignore / .gitattributes
 FormGitIgnore / FormGitAttributes Done In-app editors for repo-level ignore and attributes files. Convenience over opening them in an external editor.
Checkout Specific Revision (Detach HEAD)
 FormCheckoutRevision Done Explicitly checkout a commit hash and enter detached HEAD mode. Currently no UI path for this — only branch checkout exists.
3 Advanced — Power-user / niche features Used occasionally; full parity with GitExtensions requires these
Feature GitExtensions Form Status Notes
Search Commits
 SearchWindow Done LogFilter Advanced panel adds -S / -G pickaxe and path filter. Jump-to-result highlighting still TBD.
Worktrees
 WorktreeDialog Done git worktree add/list/remove. Work on multiple branches simultaneously without stashing. Increasingly common workflow.
Archive
 FormArchive Done git archive — export tree at any ref as .zip or .tar.gz.
Repository Verify / fsck
 FormVerify Done git fsck — find dangling/lost objects, broken refs, corrupted objects.
GPG Commit Signing
 RevisionGpgInfoControl Done Show GPG signature status per commit (✓ verified / ✗ invalid / unsigned). Sign commits via -S flag. Required by some orgs for compliance.
Mailmap Editor
 FormMailMap Done Edit .mailmap to canonicalize author names/emails across history. Niche but useful for repos with contributor name inconsistencies.
Settings / Preferences
 FormSettings Partial Git config (local + global) editor with common keys: user name/email, signing key, sign-by-default, editor, autocrlf, pull.rebase, diff/merge tool. Theme is wired separately. Deferred: SSH key generator, commit-template editor, hotkey rebinder.
Sparse Checkout
 FormSparseWorkingCopy Done git sparse-checkout — check out only a subset of the working tree. Useful for huge monorepos.
Custom Scripts / Hooks UI
 UserScriptContextMenuExtensions Deferred User-defined commands added to context menus. Deferred — script runtime + shell-injection safety + persistence + per-context substitution is a self-contained day's work; power users can drop to the terminal in the meantime.
GitHub / GitLab Integration
 RepoHosting Deferred Create pull requests, browse issues, fork repos directly from the app. Deferred — full OAuth + REST API per provider is a multi-day project on its own.
